# 📊 Task 02: Employee Attrition Analysis Dashboard

## 📌 Objective

The objective of this task is to analyze employee attrition data and create an interactive dashboard to understand the factors affecting employee turnover.

---

## 🛠️ Tools Used

* Power BI 📊
* Microsoft Excel / CSV Dataset
* Visual Studio Code (for documentation)

---

## 📂 Dataset

* Dataset: Employee Attrition Dataset
* Contains details such as:

  * Employee Age
  * Department
  * Job Role
  * Salary
  * Attrition (Yes/No)
  * Years at Company

---

## 🚀 Steps Performed

### 1. Data Collection

* Imported the employee attrition dataset (Excel/CSV format)

### 2. Data Cleaning

* Removed missing values
* Checked for duplicates
* Ensured correct data types

### 3. Data Transformation

* Categorized important fields
* Created calculated columns (if needed)

### 4. Dashboard Creation (Power BI)

Created an interactive dashboard with:

* 📊 **Attrition Count**
* 📉 **Attrition by Department**
* 📈 **Attrition by Age Group**
* 💰 **Attrition by Salary Range**
* 🧑‍💼 **Attrition by Job Role**

### 5. Insights Generation

* Identified departments with high attrition
* Observed trends based on age and salary
* Analyzed employee retention patterns

---

## 📊 Dashboard Features

* Interactive filters (slicers)
* Clear visualizations (bar charts, pie charts, cards)
* Easy-to-understand layout

---

## ▶️ How to Use

1. Download the `.pbix` file from this repository
2. Open it using Power BI Desktop
3. Interact with filters and visuals to explore insights

---

## 📌 Key Insights

* Certain departments show higher attrition rates
* Employees with lower salary ranges tend to leave more
* Early-career employees have higher attrition
